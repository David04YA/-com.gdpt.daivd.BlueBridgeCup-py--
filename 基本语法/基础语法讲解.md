 函数的定义与使用

 定义函数 def fun_name(arg):
返回值:return value
默认参数： def fun_name(arg=10):


可变参数：def func(*args)或者def func(**kwargs)

可变参数允许函数接受任意多的参数，分为两种形式
*args：接受任意数量的位置参数，将它们打包成一个元组。
**kwargs：接受任意数量的关键字参数，将它们打包成一个字典。 
元组是序列，字典是键值对
